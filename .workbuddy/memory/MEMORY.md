# 项目长期备忘 · Teen-AI-Edu

## 项目定位
- AI 青少年培训（4–6 年级 / 10–12 岁为主）。当前自做课件：生图提示词课、水果忍者 AI 编程课。
- 团队是**程序员、无授课经验**；线下免费培训以**攒教学经验**为目标，**不追转化**。成功标准=流程跑通+经验沉淀。
- 最大风险是"讲不好/控不住场"，不是护城河。给课件要重"可落地 + 控场实操"，别只给能跑的代码。

## 课件组织约定（用户要求）
- **一个课件一个文件夹**，直接平铺在项目根，与 `儿童AI故事视频/` `官方文章和政策/` 同级；不套额外父容器。
- 每份课件至少含：可执行产物 + `教案.md`。进阶再配 `教师授课手册.md` `学生任务卡.md`。

## 专业授课标准（用户明确期望）
- 用户要求课件达到**专业线下授课**水准：**生态课堂**（心理安全/真实产出/协作）、**完整师生互动**（话术/信号/分层/互评）、**控场实操**（口令/翻车救场/复盘）。
- 推荐范式：5E（Engage→Explore→Explain→Elaborate→Evaluate）+ 做中学 + 形成性评价；目标写可观测行为；引用已知调研（儿童AI认知"高渗透·浅理解·弱批判"）支撑设计。
- 程序员老师视角：给"到现场动手做的清单"，少给理论。

## 儿童 AI 认知调研结论（2025–2026，写课件可用）
- 高渗透（80%+ 用过）、浅理解（近半数认不出 AI 幻觉）、弱批判、有依赖苗头（小学 21.2% 不想自己思考）。
- 教学暗示：练"提清楚需求 + 判断对不对 + 自己做主"；借机点"AI 也会错，要自己运行验证"。

## 水果忍者 AI 编程课（手势版）要点
- 切割=摄像头手势(MediaPipe Hands 认食指指尖, landmark 8)；鼠标/触屏备用。手势需 `http://localhost`(python -m http.server 8000)+摄像头+联网。
- 第一版用鼠标保跑通，手势作模块三升级挑战。三个旋钮：GRAVITY/SPAWN_INTERVAL/BOMB_CHANCE。
- 文件：`水果忍者AI编程课/` 下 index.html / style.css / game.js / 教案.md / 教师授课手册.md / 学生任务卡.md。

# 儿童 AI 故事视频

把孩子的一张手绘原画变成一个有配图、有孩子声音、可以发到微信的 3:4 故事视频。

整个过程由 WorkBuddy、TRAE Work 或其他支持 Agent Skills 的工具执行，不需要使用剪映，也不调用动态视频模型。

## 孩子需要做什么

1. 画一个喜欢的角色或物品。
2. 回答几个简单问题，共同决定故事。
3. 录制 15-30 秒清晰声音。
4. 选择喜欢的故事方向和最终配图。

其余步骤由 Skill 自动完成：整理原画、生成故事配图、克隆声音、加入字幕并输出 MP4。

## 文件夹说明

| 文件夹 | 放什么 |
|---|---|
| `input/` | 本次要使用的原画和参考声音（位于工作区根目录） |
| `项目名/` | 每个孩子生成的独立文件夹（生成在工作区根目录） |
| `脚本/` | 自动生图、配音、合成和检查程序（含单元测试） |
| `参考/` | 视频创作流程、故事规则、技术和隐私说明 |
| `样例/` | 不含真实儿童信息的示例故事 |
| `资源/` | 可选的片头、字体和其他公共素材 |
| `评估/` | 用于检查 Skill 是否容易正确触发的测试任务 |

第一次使用，只需要阅读本文件。Agent 会根据任务自动读取其他说明。

## 第一次准备

1. 确保电脑已安装 Python 3.10 或更高版本，以及 FFmpeg。
2. 复制 `.env.example` 为 `.env`。
3. 在 `.env` 中填写 `VOLCENGINE_API_KEY` 和 `MIMO_API_KEY`。
4. 不要把 `.env` 发给其他人，也不要把密钥直接写进故事文件。

运行环境检查：

```bash
python3 脚本/run.py preflight
```

## 最简单的制作方式

通常不需要自己输入命令。把这个文件夹作为 Skill 导入 WorkBuddy 或 TRAE，然后说：

> 请用儿童 AI 故事视频 Skill，把这张原画做成一个故事。先和孩子一起确定故事，再使用这段声音配音。

如果需要直接运行：

```bash
python3 脚本/run.py all \
  --project 我的故事 \
  --drawing input/原画.jpg \
  --voice input/声音.m4a \
  --story 参考/story.example.json
```

执行中断后再次运行同一命令，会跳过已经完成的文件。

## 最终产物

每个项目中会保留：

- `final/故事视频.mp4`：可以直接分享的 1080×1440 视频。
- `review/故事板.jpg`：四张配图的预览总图。
- `text/story.json`：完整故事和每幕内容。
- `images/`：角色标准图和每幕配图。
- `voice/`：每幕旁白。
- `verification.json`：自动检查结果。

## 未成年人声音

声音克隆会把参考录音发送给小米 MiMo 服务。正式课堂使用前，应获得监护人明确同意。默认情况下，Skill 在视频成功后删除为了上传而生成的标准化声音副本；原始录音是否保留由老师自行决定。

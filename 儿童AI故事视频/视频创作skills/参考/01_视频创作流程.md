# 视频创作执行流程

## 核心制作步骤

视频创作应遵循以下核心步骤以确保顺利生成：

1. **环境检查**：运行 `python3 脚本/run.py preflight` 验证环境与 API Key 配置。
2. **项目初始化**：使用独立编号或昵称建立项目（不使用真实姓名），导入原画和音频。
3. **确定故事**：编写符合规则的四幕故事，并保存为项目的 `text/story.json`。
4. **生成配图**：运行图片生成命令，基于原画和生成的角色标准图渲染四幕 3:4 故事配图。
5. **声音克隆**：处理 15-30 秒参考音频并上传生成每幕旁白。
6. **视频合成**：合成最终的 3:4 故事视频并完成兼容性校验。

## 制作要点

- **项目隔离**：每个视频创作项目使用唯一的编号或昵称建立，避免文件或配置重名覆盖。
- **并发控制**：当有多个项目同时进行创作时，应保持较低的图片和声音 API 请求并发，防止触极限流导致请求失败。
- **隐私与安全**：禁止在项目名称、故事文本、提示词等处包含全名、学校、住址等真实隐私信息。首发前请由监护人进行内容检查。

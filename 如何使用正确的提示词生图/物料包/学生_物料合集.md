# 学生用 · 物料合集（可打印）

> 打印建议：A5 或 A4 彩色，每人一份。低龄孩子用"提示词积木贴纸"辅助填卡。

---

## ① 画面五要素填空卡

```
我想画的是：__________________________（主体）

它正在：____________________________（场景 / 在做什么）

画成这种感觉：______________________（风格：水彩 / 卡通 / 像素 / 绘本…）

光是这样的：________________________（光线：暖黄 / 清晨 / 星空 / 夕阳…）

整体情绪是：________________________（情绪：开心 / 勇敢 / 温柔 / 神秘…）
```

> 拼起来就是你的提示词！没有标准答案，只有"更清楚"。

---

## ② 清楚 vs 模糊对照卡

| 模糊（AI 听不懂） | 清楚（AI 会画） |
|---|---|
| 画只猫 | 一只趴在窗台的橘猫，午后阳光 |
| 画个好看的场景 | 秋天的树林小路，落叶满地，金色光线 |
| 一只巨大的迷你小猫 | 一只迷你小猫蹲在大鞋子里 |
| 一个勇敢的骑士 | 一个勇敢的骑士，绘本插画风格，冷色调 |

## ③ 等待时任务卡（轮流生图时用，别闲着）

**改词卡**：把"坏话"改成清楚的（写下来）
- 「画个酷的机器人」→ ________________
- 「美丽的风景」→ ________________
- 「一只很大很小的龙」→ ________________

**挑错卡**：看这张 AI 图，找它画错的地方（打勾）
- [ ] 手指/手脚数量不对
- [ ] 有违反物理的地方（水往上流？悬浮？）
- [ ] 图里有乱七八糟的"字"
- [ ] 和上一张比，角色长得不一样了
- 它错在哪：______________

```

├─────────────────────────────┤
│                             │
│      [这里放你的图 3:4]       │
│                             │
├─────────────────────────────┤
│ 我的提示词：_________________ │
│                             │

```

> 感悟写啥都行，比如："我把'好看'换成了'金色光线'，AI 就画对了。"

## ⑥ 灵感图卡使用说明（卡住时看）

想不出画啥？挑一张灵感图卡，做两步：
1. **描述**：用五要素把它说出来（主体/场景/风格/光线/情绪）。
2. **变体**：改其中一块（换个主体、换个场景），让 AI 画一张"像但又不一样"的。
> 注意：不是抄它，是拿它当跳板。

# 官方预置声音参考

本目录存放了小米 MiMo 官方内置音色的声音样品（16-bit WAV，24kHz 单声道）。

> [!WARNING]
> **不要**将本目录下的官方声音文件复制到项目的 `输入/` 文件夹下并命名为 `声音参考.wav`！这会导致程序错误地对其进行二次克隆，从而浪费调用额度并降低合成音质。
> 官方预置声音直接由小米官方模型生成，不需要进行声音克隆。

## 预置音色列表

*   **冰糖_小孩女.wav**：清澈甜美的明亮女声。
*   **苏打_小孩男.wav**：阳光活力的明亮男声。
*   **茉莉_成年女.wav**：温柔知性的成熟女声。
*   **白桦_成年男.wav**：沉稳磁性的成熟男声。

## 如何使用和修改音色
1. **优先使用创作者授权的声音**：如果在项目 `输入/` 下放置了创作者自己录制并授权使用的声音文件，并命名为 `声音参考.wav`（或 `.mp3`/`.m4a`），程序将自动使用**声音克隆**模式。
2. **使用官方内置音色**：若不提供个性化声音参考，程序将默认使用官方内置音色。您可以通过以下两种方式指定使用哪一个官方音色：
   * **方法 A（Word 文档修改）**：直接在 `故事.docx` 的 `配音设定` 部分，修改 `★ 配音音色` 字段（例如：`★ 配音音色：苏打`）。
   * **方法 B（全局修改）**：在 `config.json` 的 `voice.preset_voice` 字段中进行全局配置（如 `"preset_voice": "冰糖"`）。

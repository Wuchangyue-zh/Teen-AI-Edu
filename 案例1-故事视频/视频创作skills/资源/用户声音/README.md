# 用户自备声音参考

本目录用于存放用户（如学生、创作者）自己录制的声音参考音频文件。

## 录音建议与要求
为了获得更好的声音克隆效果，建议用户录制的声音文件满足以下要求：
1. **时长**：**15~30秒**。
2. **环境**：选择安静的室内录制，**绝对不能有任何背景音乐**、嘈杂声或旁人说话声。
3. **说话方式**：用自然的普通话讲故事语气进行录制，嘴巴靠近麦克风，音量适中。

## 如何在项目中使用
1. 将录好的音频文件（支持 `.wav`、`.mp3`、`.m4a` 等格式）保存在此目录下，方便统一管理。例如命名为 `小明_声音参考.wav`。
2. 制作故事视频时，将该文件复制到具体项目的 `输入/`（或 `input/`）文件夹下，并重命名为 `声音参考.wav`（或保持原格式扩展名，如 `声音参考.mp3`）。
3. 脚本在执行声音生成时，如果检测到项目 `输入/` 下有 `声音参考.*` 文件，会自动调用小米 MiMo 的克隆模型对该音色进行复刻。

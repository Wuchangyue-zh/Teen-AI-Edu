# 周一续命咖啡

本次按用户明确授权连续执行，用于测试三步流程、短文字和纯白背景 GIF。

- **聊天场景**：周一早晨靠咖啡强撑，结果一看到“开会”又瞬间没电；适合回复“我先喝口咖啡”。
- **角色**：原创圆滚滚小树懒职员，浅蓝衬衫、深色领带、黑眼圈和一只咖啡杯。
- **文字设计**：每格最多 3 个字，依次为“续命中”“满血！”“开会？”“没电了”，文字放在角色上方的留白处，动作仍能独立表达笑点。

1. **铺垫**：小树懒趴在办公桌上，捧着咖啡，头顶写“续命中”。
2. **触发**：喝一口后眼睛发亮，满血弹起，头顶写“满血！”。
3. **包袱**：电脑弹出一个简单会议图标，小树懒僵住，头顶写“开会？”。
4. **循环**：能量条瞬间清空，小树懒又趴回原来的咖啡杯旁，头顶写“没电了”；姿势接回第一格。

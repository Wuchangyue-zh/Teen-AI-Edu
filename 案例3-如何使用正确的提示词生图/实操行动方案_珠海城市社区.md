# 实操行动方案 · 珠海城市社区首场试跑

> 对象：珠海街道社区 4–6 年级孩子 + 城市家长｜入口：社区党群服务中心/活动室
> 前提调整：不考虑农村人群。城市家庭设备条件较好、家长识货且 skeptical——**护城河质疑上升为头号风险**，设备问题靠 BYOD 部分化解。
> 目标：首场不求转化，先**跑通流程 + 拿"家长眼睛亮"和"进群率"**，验证教案可落地性。

---

## 一、场景定准（珠海城市社区）

| 维度 | 假设与对策 |
|---|---|
| 场地 | 社区活动室，有 WiFi 但带宽有限 → **控并发、备预生成图库** |
| 设备 | **BYOD 为主**（家长让孩子带手机/平板）+ 公备 2–3 台平板兜底 |
| 人数 | 首场 **15 人**（小规模试跑，便于控场与观察） |
| 师生比 | 1 主讲 + 1–2 助教（助教巡场：帮翻车、控设备、盯内容安全） |
| 家庭 | 城市家长，时间紧、识货、怕被套路 → **护城河话术 + 能力证据卡 + 不硬推销** |
| 孩子 | 城市孩子更可能已用过 AI（调研：城市 63.7%）→ **分层任务，防"太浅"** |

---

## 二、针对差评的实操对策（怎么改、怎么做）

### P0-1 作品单薄 → 升级为"图文小卡"
- **做法**：默认作品 = 1 张主图 + 孩子写的"我的提示词 + 一句感悟" + 统一片头"珠海XX社区·AI 小创客·XXX 作品"，用 Canva/剪映模板 5 分钟拼成可分享卡片图。
- **为什么**：单张 AI 图太单薄、不够"眼睛亮"；图文小卡让作品有"包装"、家长转发有面子，且保留孩子原创痕迹（提示词+感悟是孩子的，护城河）。
- 三连图作为"挑战任务"留给高频用户，不设为默认（控时间）。

### P0-2 生图等待/设备 → BYOD + 分组轮换 + 等待时任务
- **分组轮换**：3–4 人一组共用 1–2 台设备，组内轮换"生成"与"纸上任务"。
- **等待时任务**（永远不空等）：排队时做"纸上改词卡 / 挑错卡 / 写下一个灵感"——这些本身也是教学。
- **预生成图库**：课前用统一账号生成 **20 张合格图**存本地，网络/额度崩时直接调用做"挑图 + 反推提示词"，流程不断。
- **控并发**：错峰提交，分 3 批每批 5–6 人同时生，主讲控节奏；豆包（主）+ 即梦（备）双账号。

### P0-3 护城河质疑 → 开场 + 首映两段话术 + 能力证据卡
- **开场 30 秒话术**（正面回答"凭什么来上课"）：
  > "在家用豆包是'会用'，我们今天教的是'会想 + 会判断'——把脑子里的画面说清楚、还要会挑 AI 的错。这是在家自己玩不出来的，也是同伴互评才有的成长场景。"
- **首映话术**：让作品和能力证据卡说话，不硬推销正式课。
- **能力证据卡**（发给家长）：列孩子今天练了什么——"表达精确性 / 批判检查 / 迭代思维"，让价值**可衡量**、可转发。

### P1-4 3 小时太长 → 压到 2.5 小时
- 合并休息、压缩理论演示至 15 分钟、作品墙与首映合一。中段用"生图等待 + 游戏"交替，不让孩子干坐。

### P1-5 没讲伦理/隐私 → 加 10 分钟"AI 小规矩"
- 嵌在"看图挑错"后：①不画别人脸、不造假图 ②AI 会出错别全信 ③作品分享前先问家长。
- 城市家长对隐私敏感（调研：47.4% 孩子担心隐私），这反而是**加分项/卖点**。

### P1-6 翻车挫败 → 翻车即教学 + 保底图
- 翻车不删，当"找 bug 游戏"，奖励"找出最多 bug"的孩子。
- 每人保底有一张能用的图（预生成图库兜底 / 老师帮调），不让任何孩子空手/丢脸。

### P1-7 对已会用孩子太浅 → 分层挑战卡
- 高频用户发"挑战卡"：三连图小故事 / 角色一致性 / 复杂构图，避免"太浅"。

### P2-8 教师能力 / 翻车 → 课前预演 + 话术卡
- 课前教师跑通样片；**三档对比演示用预生成结果**（不依赖现场网络演示，避免翻车）。
- 助教兜底现场翻车；后续可补教师逐字稿。

### P2-9 "免费课=套路" → 坦诚口径
- 开场明说："今天公益体验，后续有正式课，先体验再决定，不强求。"用作品说话，不硬推。

## 三、首场试跑最小可行清单（MVP of MVP）

| 项 | 配置 |
|---|---|
| 人数 | 15 人（4–6 年级） |
| 时长 | 2.5 小时 |
| 设备 | BYOD + 3 台公备平板 + 1 主讲 1 助教 |
| 工具 | 豆包（主）+ 即梦（备），双账号，额度预充 |
| 预生成 | 20 张保底图 + 3 档对比演示图 + 4 张翻车图（课前存本地） |
| 物料 | 五要素卡、灵感图卡 6 张、积木贴纸、图文小卡模板、能力证据卡、家长须知 |
| 流程（压缩版） | 破冰 → 分层开场+反护城河话术 → 原理+三档对比 → 五要素+找错 → 看图挑错+AI 小规矩 → 实践1（分组轮换+等待任务）→ 实践2 迭代 → 图文小卡成片 → 首映+能力证据卡 |

## 四、试跑要看什么数（对齐项目核心指标）

项目核心指标 = **免费→付费转化率**。但首场不求转化，先验证流程与拿信号：

| 指标 | 目标 | 说明 |
|---|---|---|
| 私域沉淀率 | ≥ 90% | 到场家长加微信/进群（入口即沉淀） |
| 首映"眼睛亮"比例 | 主观观察 | 家长看到图文小卡的反应，记录"明显点赞/拍照/转发"的比例 |
| 课后 3 天咨询数 | 信号 | 私聊问正式课的家长数（转化前置信号） |
| 孩子 NPS | "还想再来"比例 | 课后问孩子愿不愿意再来 |
| **流程崩点记录** | 必做 | 设备/等待是否崩、翻车率、注意力掉点时刻、护城河质疑被问几次、话术是否顶住 |

> 首场跑完，按"崩点记录"改 v3，再放大到第 2、3 场；连续 2 场流程不崩 + 进群率≥90% + 有家长主动问正式课，才算具备铺开资格。

---

## 五、一句话执行原则

> **首场 15 人、2.5 小时、BYOD+预生成兜底；作品升图文小卡、开场讲清护城河、加 10 分钟 AI 小规矩；不求转化，先跑通流程、拿进群率和"眼睛亮"信号。**

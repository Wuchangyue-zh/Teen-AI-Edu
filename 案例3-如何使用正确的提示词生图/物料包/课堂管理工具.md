# 课堂管理工具（可打印）

> 给非教师团队用：控场、控时、控分组、做复盘。

---

## 一、课堂口令（定一个专注信号）

| 信号 | 教师说 | 孩子做 |
|---|---|---|
| 专注 | 「3——2——1」 | 屏幕朝下，看老师 |
| 安静 | 「耳朵——」 | 「准备好了」 |
| 开始生图 | 「导演就位——开始！」 | 点生成 |
| 停手 | 「屏幕朝下」 | 设备扣桌 |

> 比喊"安静"管用。课前先和孩子练 2 次。

---

## 二、分组轮换表（示例：15 人 / 4 组）

| 组 | 成员（昵称） | 设备 | 第1轮 | 第2轮 | 第3轮 |
|---|---|---|---|---|---|
| A | 4 人 | 设备1（项目账号A） | 生图 | 等待任务卡 | 互评 |
| B | 4 人 | 设备2（项目账号B） | 等待任务卡 | 生图 | 互评 |
| C | 4 人 | 设备3（项目账号C） | 互评 | 等待任务卡 | 生图 |
| D | 3 人 | 公备平板 | 生图 | 互评 | 等待任务卡 |

> 规则：生图的人操作；等待的人做改词卡/挑错卡；互评的人帮生图的人检查提示词。
> 备 2–3 个项目账号分给各组，防多端登录风控。

---

## 三、计时 / 进度表（2.5 小时，投屏倒计时）

| 时间 | 环节 | 时长 | 备注 |
|---|---|---|---|
| 0:00–0:10 | 签到破冰 | 10 | 家长选择作品私发/群发方式；不强制进群 |
| 0:10–0:22 | 开场分层切入 | 12 | |
| 0:22–0:42 | 原理 + 三档对比 | 20 | 用预生成图 |
| 0:42–1:00 | 五要素 + 找错游戏 | 18 | |
| 1:00–1:13 | 看图挑错 + AI 小规矩 | 13 | |
| 1:13–1:23 | 休息 | 10 | |
| 1:23–1:50 | 实践 1 生图（分组轮换） | 27 | 等待任务卡 |
| 1:50–2:13 | 实践 2 迭代 | 23 | |
| 2:13–2:30 | 图文小卡 + 首映 | 17 | |

> 每环节倒计时投屏，主讲和孩子都看时间。超时立刻砍"三连图/挑战卡"，保单图+首映。

---

## 四、教研复盘表（每场后 30 分钟填）

| 维度 | 观察 | 下次改进 |
|---|---|---|
| 时间 | 哪段超时 / 空等 | |
| 孩子投入 | 注意力掉点时刻 | |
| 翻车 | 翻了几次、怎么处理 | |
| 讲解 | 哪句孩子没懂 / 术语 | |
| 设备 / 流程 | 崩点 | |
| 自己感受 | 哪里最慌 / 最顺 | |
| 经验沉淀 | 3 条带走的教训 | |

> 如需录音，须事先单独说明用途和保存期限并取得监护人同意；未同意时只做匿名书面复盘。

---

## 五、首场成败自检（跑完打勾）

- [ ] 流程从头跑到尾没崩
- [ ] 每个孩子至少生出 1 张图、做了图文小卡
- [ ] 没有孩子空等超过 5 分钟（等待任务卡顶住了）
- [ ] 没出现敏感内容 / 账号泄露
- [ ] 家长作品接收方式选择完成率 ≥ 90%
- [ ] 首映时至少一半家长有"眼睛亮"反应（拍照/点赞/转发）
- [ ] 填完了复盘表、有 3 条教训

> 连续 2 场勾上 ≥6 项，才算具备铺开资格。
